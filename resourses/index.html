Введення

Проект «Відкритий бюджет» надає можливість розробникам використовувати відкриті бюджетні дані при розробці власних додатків. Доступ надаєтся за допомогою REST API. Подробиці та деталі доступу до окремих ресурсів ви знайдете нижче.

Регіони

Перелік суб'єктів бюджетування за територіальним поділом. Суб'єкти ідентифікуються наступним чином: 0 — Україна, 1-24 — області.

Адреса ресурсу

GET https://api.open-budget.org/v1/areas.json

Параметри запиту

id опціонально Унікальний ідентифікатор регіону. Допускається передача декількох ідентифікаторів (див. приклад).
search_query опціонально Пошук за частиною назви регіону

Документи

Законодавчі акти в котрих закріплено поділення надходжень та витрат державного та місцевих бюджетів

Адреса ресурсу

GET https://api.open-budget.org/v1/documents.json

Параметри запиту

area_id опціонально Идентифікатор регіону, за яким робиться запит. 0 — Україна, 1-24 — області.
Докладніше про регіони дивіться тут. Допускається перечада декількох ідентифікаторів.
id опціонально Унікальний ідентифікатор документу. Допускається передача декількох ідентифікаторів.
year опціонально Рік набуття чинності документу. Допускається передача декількох ідентифікаторів.
search_query опціонально Пошук входження значення параметру у назву документу

Витрати

Статті витрат державного та обласних бюджетів, зґруповані за регіонами та роками

Адреса ресурсу

GET https://api.open-budget.org/v1/expenses.json

Параметри запиту

area_id опціонально Вказує на ідентифікатор регіону, по котрому потрібно зробити виборку. 0 — Україна, 1-24 — області.
Докладніше про регіони дивіться тут. Допускається перечада декількох ідентифікаторів.
year опціонально Рік, на котрий запланована певна стаття видатків. Допускається перечада декількох ідентифікаторів.
document_id опціонально Ідентифікатор документу, в якому задекларована певна стаття видатків.
Докладніше про документи дивіться тут. Допускається передача декількох ідентифікаторів.
code опціонально Код статті видатків за тимчасовою класифікацією. Допускається передача декількох ідентифікаторів.
id опціонально Унікальний ідентифікатор статті видатків. Компонується наступним чином: YYYY-RR-KK, де YYYY - рік, на який запланована стаття видатків, RR - ідентифікатору регіону, KK - код статті за тимчасовою класифікацією видатків. Наприклад: стаття з кодом ТКВ № 091206, запланована на 2014 рік у обласному бюджеті Вінницької області буде мати унікальний ідентифікатор 2014-1-91206
search опціонально Пошук за назвою статті витрат
